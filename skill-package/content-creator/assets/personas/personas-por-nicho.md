## COMO USAR ESSAS PERSONAS NA ESTRATEGIA DE VENDA DA MENTORIA

### Para conteudo organico:
1. Crie conteudo que fale das dores especificas de cada persona
2. Use as frases exatas que eles dizem (validacao emocional)
3. Mostre cases de pessoas similares que ja fizeram a transicao
4. Quebre as crencas limitantes uma a uma com conteudo estrategico

### Para copy de vendas:
1. Use o Job To Be Done funcional como headline principal
2. Use o Job emocional como subtexto que gera conexao
3. Enderece cada objecao diretamente na pagina de vendas
4. Inclua prova social especifica para cada tipo de profissional

### Para segmentacao de anuncios:
1. Crie audiencias por profissao (medicos, advogados, etc)
2. Use as palavras-chave que cada persona usa
3. Retargeting para quem engajou com conteudo especifico

### Para a jornada de compra:
1. TOFU: conteudo que desperta o problema (teto de faturamento, dependencia de presenca)
2. MOFU: conteudo que mostra o caminho (como profissionais consolidados estao fazendo a transicao)
3. BOFU: conteudo que gera decisao (cases, depoimentos, webinar estrategico, call 1:1)
