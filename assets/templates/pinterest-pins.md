## TIPOS DE PIN E ESPECIFICACOES

### Standard Pin (Pin Estatico)
| Spec | Valor |
|------|-------|

| Spec | Valor |

## OTIMIZACAO DE TITULO (100 CARACTERES)

### Estrutura do Titulo
```
[PALAVRA-CHAVE PRINCIPAL] + [BENEFICIO/COMPLEMENTO]
```

### Formulas de Titulo
```
1. "[N] [COISAS] de [TEMA] para [BENEFICIO]"
   Exemplo: "15 Ideias de Decoracao para Quartos Pequenos"

2. "Como [RESULTADO] com [METODO/FERRAMENTA]"
   Exemplo: "Como Organizar a Cozinha com Cestos de Bambu"

3. "[TEMA]: Guia Completo para [PERSONA]"
   Exemplo: "Receitas Fit: Guia Completo para Iniciantes"

4. "[RESULTADO] em [TEMPO] - [METODO]"
   Exemplo: "Sala Renovada em 1 Fim de Semana - DIY Facil"

5. "O Melhor [ITEM] para [SITUACAO/PERSONA]"
   Exemplo: "O Melhor Planner para Maes Empreendedoras"

6. "[TEMA] [ANO]: [TENDENCIA/NOVIDADE]"
   Exemplo: "Tendencias de Moda 2026: Cores e Estilos"
```

### Boas Praticas do Titulo
```
- Palavra-chave no INICIO do titulo
- Nao usar clickbait (Pinterest penaliza)
- Ser descritivo e especifico
- Incluir numeros quando possivel
- Evitar emojis no titulo (diferente de outras redes)
- Pensar em como a pessoa BUSCARIA esse conteudo
```

## DESCRICAO COM KEYWORDS (500 CARACTERES)

### Estrutura da Descricao
```
[FRASE PRINCIPAL com palavra-chave principal].
[DETALHAMENTO com palavras-chave secundarias].
[BENEFICIO para o usuario].
[CTA - o que fazer com o pin].
```

### Exemplo de Descricao
```
Ideias de decoracao para quartos pequenos que maximizam
o espaco sem gastar muito. Aprenda tecnicas de organizacao,
escolha de moveis compactos e paleta de cores que ampliam
o ambiente. Perfeito para apartamentos e kitinets.
Salve este pin para sua proxima reforma!
```

### Checklist da Descricao
```
- [ ] Palavra-chave principal na primeira frase
- [ ] 2-3 palavras-chave secundarias naturalmente inseridas
- [ ] Linguagem natural (nao parecer spam)
- [ ] Beneficio claro para quem le
- [ ] CTA (salvar, clicar, experimentar)
- [ ] Entre 150 e 500 caracteres
- [ ] Sem hashtags excessivas (maximo 2-3)
```

- Pin mais representativo
- Visual atraente e coerente
- Atualizar periodicamente

ORGANIZACAO:
- Minimo 10 pins por board antes de tornar publico
- Manter boards ativos (pinar regularmente)
- Reorganizar a ordem dos boards por prioridade

## SEO PARA PINTEREST

### Pinterest como Mecanismo de Busca
```
Pinterest NAO e uma rede social tradicional.
Pinterest e um MECANISMO DE BUSCA VISUAL.

Isso significa:
- Conteudo tem vida util LONGA (meses/anos)
- Keywords importam MAIS que hashtags
- Descricoes sao indexadas como texto
- Alt text e lido pelo algoritmo
- Nomes de arquivos de imagem importam
- Boards funcionam como categorias de SEO
```

### Pesquisa de Palavras-Chave
```
1. BARRA DE BUSCA DO PINTEREST
   - Digite sua keyword e veja sugestoes automaticas
   - Essas sugestoes = termos mais buscados

2. PINTEREST TRENDS (trends.pinterest.com)
   - Identifique termos em crescimento
   - Compare volume de busca
   - Planeje conteudo sazonal

3. GUIDED SEARCH (tags coloridas)
   - Apos uma busca, veja as tags relacionadas
   - Use como keywords secundarias

4. PINTEREST ADS KEYWORD TOOL
   - Mesmo sem anunciar, use para pesquisa
   - Mostra volume de busca estimado
```

### Onde Colocar Keywords
| Local | Prioridade | Exemplo |
|-------|-----------|---------|
| **Titulo do Pin** | ALTA | "Receitas fit para emagrecer" |
| **Descricao do Pin** | ALTA | Texto natural com keywords |
| **Alt Text da Imagem** | MEDIA | Descricao acessivel com keyword |
| **Nome do Board** | ALTA | "Receitas Fit e Saudaveis" |
| **Descricao do Board** | MEDIA | Keywords relacionadas |
| **Nome do Perfil** | ALTA | "Maria | Receitas Fit" |
| **Bio do Perfil** | MEDIA | Keywords do nicho principal |
| **Texto no Pin** | ALTA | Keyword visivel na imagem |
| **Nome do Arquivo** | BAIXA | receitas-fit-rapidas.jpg |

## DESIGN DE PINS - BOAS PRATICAS

### Elementos Essenciais
```
1. TITULO LEGIVEL
   - Fonte grande e bold
   - Maximo 8 palavras
   - Contraste com o fundo
   - Legivel em tela pequena (celular)

2. IMAGEM DE QUALIDADE
   - Alta resolucao
   - Cores vibrantes (se adequado ao nicho)
   - Composicao limpa
   - Sem poluicao visual

3. BRANDING SUTIL
   - Logo pequeno (canto inferior)
   - Cores da marca consistentes
   - Fonte padrao do perfil
   - URL do site (opcional)

4. HIERARQUIA VISUAL
   - Elemento principal em destaque
   - Titulo > Subtitulo > Detalhes
   - Fluxo de leitura de cima para baixo
```

### Cores que Performam no Pinterest
```
- Imagens com cores QUENTES (vermelho, laranja) tem mais repins
- Imagens com MULTIPLAS cores dominantes performam melhor
- Evitar imagens muito escuras ou com fundo preto
- Branco/clean funciona para nichos de lifestyle e minimalismo
- Cores saturadas chamam mais atencao no feed
```

## RICH PINS - CONFIGURACAO

### Tipos de Rich Pins
| Tipo | Fonte | O que Mostra |
|------|-------|-------------|
| **Article** | Blog/site | Titulo, autor, descricao |
| **Product** | E-commerce | Preco, disponibilidade, link de compra |
| **Recipe** | Blog de receitas | Ingredientes, tempo, porcoes |

### Como Configurar
```
1. ADICIONAR META TAGS (Open Graph ou Schema.org)
   - Adicionar no <head> do seu site
   - Usar og:title, og:description, og:image
   - Para produtos: product:price, product:availability

2. VALIDAR NO PINTEREST
   - Acessar: developers.pinterest.com/tools/url-debugger/
   - Colar a URL do seu site
   - Verificar se os dados aparecem corretamente

3. SOLICITAR ATIVACAO
   - Clicar em "Apply" apos validacao
   - Aguardar aprovacao (geralmente 24-48h)
   - Uma vez aprovado, todos os pins do dominio serao Rich Pins

4. BENEFICIOS
   - Mais informacoes visiveis no pin
   - Atualizacao automatica de dados
   - Maior credibilidade e CTR
   - Selo de verificacao do site
```

## ESTRATEGIA DE HASHTAGS

### Regras de Hashtags no Pinterest
```
- Maximo: 20 hashtags (recomendado: 2-5)
- Hashtags tem MENOS peso que keywords na descricao
- Usar apenas hashtags relevantes e especificas
- Nao usar hashtags genericas (#love #beautiful)
- Hashtags funcionam como links de busca
```

### Estrutura de Hashtags
```
#[PalavraChavePrincipal] #[Nicho] #[SubTema]

Exemplos por nicho:
Culinaria: #ReceitasFaceis #CulinariaEmCasa #ReceitasFit
Decoracao: #DecoracaoDeInteriores #DIY #CasaPequena
Marketing: #MarketingDigital #EmpreendedorismoDigital #SEO
Moda: #ModaFeminina #LookDoDia #TendenciasDeModa
```

## ANALYTICS E METRICAS

### Metricas Principais
| Metrica | O que Significa | Meta |
|---------|----------------|------|
| **Impressoes** | Quantas vezes o pin apareceu | Crescimento mensal |
| **Cliques no Pin** | Cliques para ampliar | > 1% das impressoes |
| **Cliques no Link** | Cliques para o site | > 0.5% das impressoes |
| **Salvamentos** | Repins/saves | > 0.3% das impressoes |
| **Engajamento Total** | Cliques + saves | > 2% das impressoes |
| **Alcance Mensal** | Contas unicas alcancadas | Crescimento constante |
| **Audiencia Engajada** | Quem interage regularmente | Crescimento constante |

### Relatorio Mensal
```
VISAO GERAL:
- Impressoes totais: [X]
- Cliques totais: [X]
- Salvamentos totais: [X]
- Novos seguidores: [X]

TOP 5 PINS (por impressoes):
1. [Titulo] - [impressoes] - [cliques]
2. ...

TOP 5 BOARDS (por engajamento):
1. [Board] - [saves] - [cliques]
2. ...

ANALISE:
- O que funcionou: [...]
- O que nao funcionou: [...]
- Ajustes para o proximo mes: [...]
```

### Ferramentas de Analytics
```
1. PINTEREST ANALYTICS (nativo)
   - Visao geral do perfil
   - Performance por pin
   - Dados demograficos

2. PINTEREST TRENDS
   - Tendencias de busca
   - Sazonalidade de termos
   - Comparacao de keywords

3. FERRAMENTAS EXTERNAS
   - Tailwind (agendamento + analytics)
   - Canva (design + publicacao)
   - Google Analytics (trafego vindo do Pinterest)
```

## ERROS COMUNS A EVITAR

```
1. TRATAR COMO REDE SOCIAL: Pinterest e busca visual, nao feed social
2. IGNORAR SEO: Keywords sao mais importantes que hashtags
3. PINS QUADRADOS: Formato 2:3 vertical performa muito melhor
4. TEXTO ILEGIVEL: Fonte pequena demais para celular
5. SEM CONSISTENCIA: Pinar 30 pins em 1 dia e sumir por semanas
6. SEM LINK: Pin sem URL de destino nao gera trafego
7. IMAGENS RUINS: Baixa resolucao ou sem atratividade
8. BOARDS DESORGANIZADOS: Boards com temas misturados
9. SEM SAZONALIDADE: Nao planejar conteudo com antecedencia
10. COPIAR PINS: Usar imagens de terceiros sem autorizacao
```
