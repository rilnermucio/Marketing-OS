## Como Adicionar Stories

1. Defina um ID único (STORY-XXX)
2. Escreva uma descrição clara
3. Classifique o tipo (Docs, Quality, Enhancement, DevOps, Feature)
4. Estime o tamanho (P, M, G)
5. Atribua prioridade (P0-P3)
