# Ads Agent Skill

Expert paid advertising skill focused on creating high-converting ad copy that maximizes ROAS across all major advertising platforms.

## When to Use This Skill

- Writing ad copy for any paid platform
- Creating multiple ad variations for testing
- Optimizing existing ads for better performance
- Structuring campaigns and ad sets
- Writing headlines, descriptions, and primary text
- Developing creative angles for products/services
- Adapting copy for different audiences and placements

**Meta Ads Best Practices:**
- Lead with benefit, not feature
- Use emojis sparingly (1-2 max)
- Hook in first line (before "See more")
- Test video vs. static
- Use social proof when possible

## A/B Testing Framework

### What to Test in Ads

| Element | Test Variations | Impact |
|---------|-----------------|--------|
| **Hook/First line** | Different angles, questions vs. statements | High |
| **Headline** | Benefit vs. feature, length | High |
| **CTA** | Different actions, urgency | Medium |
| **Social proof** | Numbers, testimonials, logos | Medium |
| **Offer** | Price, discount, bonus | High |
| **Creative** | Video vs. image, UGC vs. polished | High |
| **Emoji** | With vs. without | Low |
| **Length** | Short vs. long copy | Medium |

### Testing Strategy

```
PHASE 1: Creative Testing
- Test 3-5 different creative concepts
- Same audience, different visuals/hooks
- Goal: Find winning creative angle

PHASE 2: Copy Testing
- Keep winning creative, test copy variations
- Test headlines, body copy, CTAs
- Goal: Optimize messaging

PHASE 3: Audience Testing
- Keep winning creative + copy
- Test different audiences
- Goal: Find best audience fit

PHASE 4: Scaling
- Increase budget on winners
- Continuous testing of new angles
- Refresh creative before fatigue
```

## Ad Metrics and Optimization

### Key Metrics by Platform

| Platform | Primary Metrics | Benchmarks |
|----------|-----------------|------------|
| **Meta** | CTR, CPC, ROAS, CPM | CTR 1-2%, ROAS 3x+ |
| **Google Search** | CTR, CPC, Conv Rate, ROAS | CTR 3-5%, Conv 2-5% |
| **Google Display** | CTR, View-through, CPM | CTR 0.5-1% |
| **TikTok** | CTR, CPM, Hook rate, ROAS | Hook rate 30%+ |
| **LinkedIn** | CTR, CPL, Engagement | CTR 0.4-0.6% |

### Optimization Signals

| Signal | Action |
|--------|--------|
| Low CTR, high impressions | Creative/copy not resonating. Test new hooks. |
| High CTR, low conversions | Landing page issue or wrong audience. |
| High CPM | Audience too competitive or ad relevance low. |
| Declining performance | Creative fatigue. Refresh ads. |
| High frequency | Audience exhaustion. Expand targeting. |
