# Video Agent Skill

Expert video content skill focused on creating scripts that capture attention, maintain retention, and drive action across all video platforms.

## When to Use This Skill

- Writing scripts for YouTube long-form videos
- Creating Reels, TikTok, or Shorts content
- Developing Video Sales Letters (VSL)
- Crafting hooks and retention strategies
- Planning thumbnail and title combinations
- Structuring educational or entertaining video content

| Instagram Reels | 15-90s | Instagram | Reach, engagement |

### Retention Strategies

| Moment | Challenge | Strategy |
|--------|-----------|----------|
| 0-3s | Scroll/skip decision | Pattern interrupt, hook |
| 3-30s | "Is this worth my time?" | Promise value, create curiosity |
| 30s-Mid | Attention fatigue | Open loops, momentum, variety |
| Mid-End | "I got what I need" | Tease what's coming, escalate value |
| Last 30s | Drop before CTA | Reward for staying, clear CTA |

### Retention Techniques

| Technique | Description | Example |
|-----------|-------------|---------|
| Open loops | Tease content coming later | "And at the end, I'll share the #1 mistake that ruins everything" |
| Pattern interrupts | Visual/audio changes | B-roll, text on screen, angle change |
| Signposting | Tell viewers where they are | "Now for step 3 of 5..." |
| Escalation | Build to bigger reveals | Save best insight for later |
| Direct address | Speak to the viewer | "If you're still watching, you're serious about this" |

## Hook Frameworks

### The Hook Formula

```
HOOK (0-3s) = Pattern Interrupt + Promise/Curiosity

Components:
1. Visual/audio that stops the scroll
2. Statement that creates curiosity or promises value
3. Immediate context (who this is for)
```

### Hook Categories

#### 1. Curiosity Hooks
```
"This changed everything about how I [topic]..."
"Nobody's talking about this, but..."
"I just discovered why [common thing] doesn't work..."
"Here's what [expert/company] doesn't want you to know..."
```

#### 2. Controversy Hooks
```
"Unpopular opinion: [bold statement]..."
"Everyone's doing [thing] wrong..."
"I'm going to say what no one else will..."
"Stop [common advice] immediately..."
```

#### 3. Result Hooks
```
"How I [achieved result] in [timeframe]..."
"This [method] got me [specific result]..."
"I went from [before] to [after] using this..."
"[Number] [result] with just [simple method]..."
```

#### 4. Tutorial Hooks
```
"Here's exactly how to [achieve thing]..."
"3 steps to [result] (that actually work)..."
"The [topic] hack that changed my life..."
"Watch me [do thing] in real time..."
```

#### 5. Story Hooks
```
"I was [relatable situation] when [unexpected thing happened]..."
"Last week, something crazy happened..."
"I need to tell you about [intriguing thing]..."
"You won't believe what I found out..."
```

## Platform-Specific Guidelines

### YouTube Long-Form

| Element | Best Practice |
|---------|---------------|
| Title | Curiosity + keyword, under 60 chars |
| Thumbnail | Face + emotion + 3 words max |
| Description | First 150 chars = hook, keywords, timestamps |
| Cards | Add at attention dips, link to relevant content |
| End screen | Last 20s, subscribe + next video |
| Chapters | Every major section, improves retention |

### YouTube Shorts

| Element | Best Practice |
|---------|---------------|
| Duration | 30-45s optimal |
| Hook | First word must grab attention |
| Text | Key words on screen |
| Format | Vertical, full screen |
| Loop | End connects to beginning |
| CTA | Subscribe reminder |

### Instagram Reels

| Element | Best Practice |
|---------|---------------|
| Duration | 15-30s for reach, up to 90s for depth |
| Hook | First frame must stop scroll |
| Audio | Trending sounds boost reach |
| Text | Captions always (80% watch muted) |
| CTA | "Save this", "Follow for more" |
| Hashtags | 3-5 relevant hashtags |

### TikTok

| Element | Best Practice |
|---------|---------------|
| Duration | 15-30s for virality, longer for authority |
| Hook | First second is everything |
| Trends | Use trending sounds/formats |
| Text | On-screen text throughout |
| Native | Avoid reposted content look |
| Engagement | Ask questions, use duets/stitches |

## Thumbnail and Title Combinations

### Thumbnail Formula

```
THUMBNAIL = Face/Reaction + Text (3 words max) + Contrast

Elements:
- Expressive face (emotion visible)
- Bold text (readable at small size)
- High contrast colors
- Clear focal point
- No clutter
```

### Title Formulas

| Formula | Example |
|---------|---------|
| Number + Result + Timeframe | "5 Habits That Made Me Productive in 30 Days" |
| How I + Achievement | "How I Got 100K Followers Without Going Viral" |
| Question + Answer Hint | "Why Your Content Isn't Growing? (The Truth)" |
| Comparison | "Cheap vs Expensive Setup: Is It Worth It?" |
| Contrarian | "Why I Stopped [Common Advice] (And What Happened)" |

### Title + Thumbnail Combos

| Title Type | Thumbnail Type |
|------------|----------------|
| Question | Confused/curious face |
| Achievement | Excited/happy face |
| Mistake/Warning | Worried/shocked face |
| Tutorial | Before/after or tool shown |
| Reaction | Exaggerated emotion |
