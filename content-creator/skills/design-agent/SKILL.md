## Typography System

### Font Categories

| Category | Characteristics | Best For |
|----------|-----------------|----------|
| **Serif** | Traditional, authoritative, readable | Editorial, luxury, long-form |
| **Sans-Serif** | Modern, clean, versatile | Tech, digital, UI |
| **Slab Serif** | Bold, sturdy, attention-grabbing | Headlines, bold statements |
| **Script** | Elegant, personal, decorative | Accents, logos (sparingly) |
| **Display** | Unique, expressive, distinctive | Headlines, branding |
| **Monospace** | Technical, modern, code-like | Tech, data, quotes |

### Typography Pairing Rules

```
HEADING + BODY COMBINATIONS:

1. Serif heading + Sans-serif body
   Example: Playfair Display + Open Sans
   Feel: Elegant yet modern

2. Sans-serif heading + Serif body
   Example: Montserrat + Lora
   Feel: Modern with classic readability

3. Same family, different weights
   Example: Inter Bold + Inter Regular
   Feel: Cohesive, professional

4. Contrasting sans-serifs
   Example: Bebas Neue + Roboto
   Feel: Dynamic, contemporary
```

### Recommended Font Pairings
