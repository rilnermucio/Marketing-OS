# Create Infoproduct

> See [CONNECTORS.md](../CONNECTORS.md) for connected services that can provide market data and design assets.

Create a complete infoproduct blueprint with curriculum design, pricing strategy, launch plan, and sales materials based on instructional design science.

## Trigger

This command is invoked when the user says `/criar-infoproduto` followed by a product type and topic, or when they ask to create an online course, ebook, membership, workshop, mentorship, or any digital educational product.

## Inputs

Gather the following information. If any required field is missing, ask the user before proceeding:

1. **Product Type** (required) — Ebook, Online Course, Membership, Workshop, Mentorship, Challenge, Digital Templates, Masterclass
2. **Topic** (required) — Main subject and specific angle
3. **Target Audience** (required) — Who this is for, their current level, and desired transformation
4. **Expertise Level** (required) — Beginner, Intermediate, Advanced, or Mixed
5. **Price Range** (optional) — Low Ticket (R$47-197), Mid Ticket (R$297-997), High Ticket (R$2.000-10.000), Ultra (R$10.000+)
6. **Timeline** (optional) — How long the student journey takes
7. **Format Preference** (optional) — Video, text, audio, hybrid, live, recorded
8. **Existing Content** (optional) — Content already created that can be repurposed

## Product Type Specifications

### Ebook

**Structure:**
- 15.000-40.000 words
- 8-15 chapters
- Each chapter: concept + examples + action steps
- Includes worksheets/checklists as appendix

**Pricing:** R$27-97 (standalone) or free lead magnet
**Timeline to create:** 2-4 weeks
**Delivery:** PDF + bonus materials

### Online Course

**Structure:**
- 4-8 modules
- 3-6 lessons per module (15-30 min each)
- Total: 8-20 hours of content
- Each lesson: Theory (30%) + Practice (50%) + Reflection (20%)

**Pricing:** R$197-2.997 depending on depth and support
**Timeline to create:** 4-8 weeks
**Delivery:** Video platform + materials + community

### Membership/Assinatura

**Structure:**
- Monthly content drops (4-8 pieces)
- Community access (exclusive group)
- Live sessions (weekly or monthly)
- Resource library (grows over time)

**Pricing:** R$47-297/month
**Timeline to create:** 2 weeks for launch, ongoing production
**Delivery:** Platform + community + live calls

### Workshop (Live or Recorded)

**Structure:**
- 2-4 hours total
- 3-4 sessions or single intensive
- Hands-on exercises during session
- Workbook/materials included

**Pricing:** R$97-497
**Timeline to create:** 1-2 weeks
**Delivery:** Live Zoom/recorded + workbook

### Mentorship/Mentoria

**Structure:**
- 3-12 month program
- Weekly/biweekly 1:1 calls (30-60 min)
- Customized action plan
- WhatsApp/Slack access for support
- Curated resources

**Pricing:** R$2.000-15.000+ total
**Timeline to create:** 1 week framework, ongoing delivery
**Delivery:** Calls + messaging + custom materials

### Challenge (5-7 Days)

**Structure:**
- 5-7 daily missions (15-30 min each)
- Community group (WhatsApp/Telegram)
- Daily live check-ins
- Quick wins designed into each day
- Pitch on final day

**Pricing:** Free-R$47 (leads to main offer)
**Timeline to create:** 1 week
**Delivery:** WhatsApp group + videos + worksheets

## Curriculum Design Framework

### Instructional Design Principles

Apply the **backward design** method:

1. **Start with transformation:** What will the student be able to DO after completing?
2. **Define milestones:** What are the 4-8 key breakthroughs along the way?
3. **Build modules around milestones:** Each module = one milestone achieved
4. **Sequence logically:** Each module builds on the previous one

### Module Architecture

```
MODULE [N]: [Milestone Name]

🎯 Objective: [Specific skill or outcome]
⏱️ Duration: [Total time]
📊 Assessment: [How to verify learning]

Lesson 1: [Foundation concept]
  → Theory: [Key concept explained]
  → Example: [Real-world application]
  → Exercise: [Hands-on practice]

Lesson 2: [Deeper dive]
  → Theory: [Advanced concept]
  → Example: [Case study]
  → Exercise: [Implementation task]

Lesson 3: [Application]
  → Theory: [Connecting to real scenarios]
  → Template: [Ready-to-use resource]
  → Project: [Apply everything learned]

✅ Module Checkpoint: [Deliverable or quiz]
```

### Cognitive Load Management

Based on Sweller's theory:
- **Chunk information:** Max 3-7 new concepts per lesson
- **Eliminate noise:** Only essential content
- **Scaffold progressively:** Simple → Complex
- **Space repetition:** Review key concepts across modules

## Pricing Strategy

### Value-Based Pricing Formula

```
Price = (Transformation Value × Perceived Probability of Success) ÷ Time to Result

Low Ticket (R$47-197):   Information → How-to knowledge
Mid Ticket (R$297-997):  Implementation → Guided execution
High Ticket (R$2k-10k):  Transformation → Done-with-you results
Ultra (R$10k+):          Partnership → Done-for-you + access
```

### Value Stack Construction

```
Core Product:                    R$ [value]
Bonus 1: [Complementary tool]   R$ [value]
Bonus 2: [Templates/resources]  R$ [value]
Bonus 3: [Community access]     R$ [value]
Bonus 4: [Live session/Q&A]     R$ [value]

Total Value:                     R$ [sum]
Your Investment:                 R$ [actual price]
Savings:                         [X]% off
```
